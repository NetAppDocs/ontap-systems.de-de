= Senden Sie das ausgefallene Teil an NetApp zurück – AFX 1K
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn eine Komponente in Ihrem AFX 1K-Speichersystem ausfällt, senden Sie das ausgefallene Teil an NetApp zurück. Siehe die  https://mysupport.netapp.com/site/info/rma["Rückgabe und Austausch von Teilen"] Seite für weitere Informationen.
