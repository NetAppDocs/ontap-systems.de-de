= Senden Sie das fehlerhafte Teil an NetApp – AFF A320
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Senden Sie das defekte Teil an NetApp zurück, wie in den mit dem Kit gelieferten RMA-Anweisungen beschrieben. Siehe die https://mysupport.netapp.com/site/info/rma["Rückgabe und Austausch von Teilen"] Weitere Informationen finden Sie auf der entsprechenden Seite.
