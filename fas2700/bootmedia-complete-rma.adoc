= Rückgabe des fehlerhaften Teils an NetApp – AFF A220 und FAS2700
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Senden Sie das fehlerhafte Teil wie in den dem Kit beiliegenden RMA-Anweisungen beschrieben an NetApp zurück.  https://mysupport.netapp.com/site/info/rma["Rückgabe und Austausch von Teilen"]Weitere Informationen finden Sie auf der Seite.

Das FAS2700-System unterstützt ausschließlich manuelle Wiederherstellungsverfahren über Bootmedien. Die automatische Wiederherstellung über Bootmedien wird nicht unterstützt.
