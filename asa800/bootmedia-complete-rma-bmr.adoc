= Senden Sie das fehlerhafte Bootmedium an NetApp - ASA A800 zurück
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn eine Komponente in Ihrem ASAF A800 System ausfällt, senden Sie das ausgefallene Teil an NetApp zurück. Siehe die  https://mysupport.netapp.com/site/info/rma["Rückgabe und Austausch von Teilen"] Seite für weitere Informationen.
